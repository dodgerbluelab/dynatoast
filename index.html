DynamicToast

- a lightweight toast system inspired by Apple's Dynamic Island

To get started, simply add these two to the html meta tag:

<!-- DynamicToast CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/dodgerbluelab/dynatoast@latest/dt.css">

<!-- DynaToast JavaScript -->
<script src="https://cdn.jsdelivr.net/gh/dodgerbluelab/dynatoast@latest/dt.js"></script>

DynamicToast Options

  1. Text: 'Your toast message here' REQUIRED
  2. Expanded: 'Addtional info here'
  3. Type:
    • 'success'
    • 'info'
    • 'warning'
    • 'error'
    • 'async'
    • 'emoji: ðŸš€'
  4. Duration: 3000 (ms) (Will fallback to 5000)
